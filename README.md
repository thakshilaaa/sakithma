

## 🧬 About This Project

> *A digital space that brings Sakithma's academic journey to life.*

I developed this portfolio for **Sakithma Rajapaksha** — a passionate Biotechnology undergraduate at SLIIT. The goal was to create a **visually immersive** experience that reflects her field of study, using a calming **blue‑and‑white** palette and a **live animated ocean wallpaper** that feels alive.
